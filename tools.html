Statistical Tools

Tool Description Cost
G*Power Tool for statistical power analysis. Free (Install)
RAWGraphs Visualization tool. Also provides good guidelines on which graph would be appropriate given specific data parameters. Free
SmartPLS UI tool for PLS-SEM. Useful for structural equation modeling. Free for students. Paid (Free for students)
DATAtab Package for surveys and statistical tests. They provide free tutorials and explanations for almost every statistical test you might need. Free / Paid
JASP Free tool for various descriptive, frequentist, and Bayesian statistical techniques. Free (Install)
MAXQDA Advanced qualitative analysis. Supports Grounded Theory. Pricey for small projects. Paid
